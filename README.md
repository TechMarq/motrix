# Motrix - Car Maintenance & Fueling Control

Motrix é um aplicativo web mobile desenvolvido especificamente para motoristas profissionais (Uber, 99, Táxi, etc.) que precisam ter controle total sobre seus custos e manutenção preventiva.

## 🚀 Funcionalidades Atuais (MVP)

- **Cadastro de Veículo**: Armazena modelo, ano e quilometragem.
- **Controle de Abastecimento**:
  - Registro de KM, litros e valor pago.
  - Cálculo automático de consumo (km/L).
  - Cálculo de custo por KM (essencial para precificar seu trabalho).
- **Gestão de Manutenção**:
  - Alertas visuais de manutenção próxima ou vencida.
  - Histórico de serviços realizados.
- **Dashboard Premium**:
  - Gráfico de gastos recentes.
  - Métricas de eficiência em tempo real.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6+).
- **Ícones**: [Lucide Icons](https://lucide.dev/).
- **Gráficos**: [Chart.js](https://www.chartjs.org/).
- **Persistência**: LocalStorage (os dados ficam salvos no seu navegador).

## 📂 Como executar

1. Localize a pasta `scratch/motrix`.
2. Abra o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Safari).
3. Recomenda-se usar o modo de inspeção do navegador e selecionar uma visualização mobile (F12 -> Toggle Device Toolbar) para a melhor experiência.
